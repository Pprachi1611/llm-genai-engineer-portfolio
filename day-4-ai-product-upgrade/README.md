# 🤖 AI Study Assistant (Day 4)

A multi-feature AI-powered CLI tool that helps users learn, summarize, and generate quizzes using a Large Language Model (LLM).

---

## 🚀 Features

### 1. 📖 Explain Mode
- Explains any topic in simple language
- Uses bullet points and real-life examples

### 2. ✂️ Summarize Mode
- Converts long text into key points
- Removes unnecessary details

### 3. 🧠 Quiz Generator
- Generates 5 MCQs from any topic
- Includes answers for self-assessment

---

## 🧠 What is an LLM?

A Large Language Model (LLM) is an AI system trained on large datasets that can understand and generate human-like text.

This project uses an LLM API to generate responses based on user prompts.
