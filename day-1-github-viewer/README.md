# 🚀 Day 1 – GitHub Profile Viewer

## 📌 Project Overview
A simple CLI-based Python app that fetches GitHub user data using GitHub API.

---

## ⚙️ Features
- Fetch GitHub user details
- Displays:
  - Name
  - Followers
  - Following
  - Public repositories
- Handles invalid usernames

---

## 🧠 Concepts Used
- Python basics
- API calls using `requests`
- JSON handling

---

## 🛠️ Setup & Run

### 1. Install dependencies
```bash
pip install -r requirements.txt
```
